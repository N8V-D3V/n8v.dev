How to report an issue

Send a note to support@n8v.dev and include the details below so we can investigate quickly.

Include your device details
  • Device model
  • iOS version
  • App name and app version if available
Describe the problem clearly
  • What happened
  • What you expected to happen
  • Whether the issue happens every time
Helpful extras

Screenshots, short screen recordings, and clear reproduction steps usually help us resolve issues much faster.

Support is currently available for The Monster Math, The Monster Match, and future N8V DEV releases.
