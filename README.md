## How To Work In This Repo

1. Read `AGENTS.md`.
2. Review the relevant COP documents in `docs/cop/`.
3. For website changes, inspect the related HTML page and shared CSS first.
4. Keep changes minimal, explicit, and consistent with the static-site structure.
5. Validate links, assets, and any affected COP artifacts before reporting completion.
